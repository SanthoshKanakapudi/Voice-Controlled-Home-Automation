#Voice Controlled Home Automation

A realtime voice controlled Home Automation project successfully demonstrated using technologies like Arduino, ESP8266 integrated with Google Assistant via IFTTT Applet webhooks, Blynk Cloud and Bluetooth control
This Project aims for Voice automation in both Offline and Online environments.
A Mini Project from E2 and E3
Software Used
1. Arduino IDE
2. VS Code for Python(We can also use Pycharm)
3. Arduino Bluetooth Control
4. Blynk Cloud
5. IFTTT
6. Google Assistant

Components used:
1. Arduino UNO
2. NodeMCU (ESP8266)
3. BreadBoard
4. Bulbs
5. 5V Relays
6. Bluetooth Module
7. Servomotor and Motor(as Fan)
8. Pins

Logic:
1. Made individual UIs using Python and HTML
2. Wifi is accessed by ESP8266
3. Can be controlled in online Mode using Blynk interface, Python UI(VOice Mode and Buttons Mode) and Google Assistant
4. Can be controlled in offline Mode using Bluetooth Application with Predefined commands!

Can be implemented as real product with a little customization and can be easily integrated using AI
